# 📇 GET INFO Microservice

This microservice is responsible for retrieving a user's information.

## 🚀 Getting Started

### Requirements

- [Node.js](https://nodejs.org/) (v16+ recommended)

---

## ▶️ How to Run

```bash
cd get-info-ms
npm install
npm start

🔄 API Endpoints
GET /api/get-info – Retrieves a user's information.
