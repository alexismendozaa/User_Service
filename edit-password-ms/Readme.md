# 🔑 EDIT PASSWORD Microservice

This microservice is responsible for updating a user's password.

## 🚀 Getting Started

### Requirements

- [Node.js](https://nodejs.org/) (v16+ recommended)

---

## ▶️ How to Run

```bash
cd edit-password-ms
npm install
npm start

🔄 API Endpoints
POST /api/edit-password – Updates a user's password.
