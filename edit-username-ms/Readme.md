# 🧑‍💻 EDIT USERNAME Microservice

This microservice is responsible for updating a user's username.

## 🚀 Getting Started

### Requirements

- [Node.js](https://nodejs.org/) (v16+ recommended)

---

## ▶️ How to Run

```bash
cd edit-username-ms
npm install
npm start

🔄 API Endpoints
POST /api/edit-username – Updates a user's username.
